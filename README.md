# vue-3d-loader 
修订版 vite+three.js

vueJS + threeJS整合的一个3d展示组件。  

支持.dae/.fbx/.gltf/.glb/.obj/.ply/.stl/.json，并支持同一个场景导入多个不同3D模型，支持mtl材质以及jpg/png等图片纹理
